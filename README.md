# 🌐 API Proxy Rotatif

Cette API retourne un proxy rotatif aléatoire au format `ip:port:user:password` à chaque requête.

## 🚀 Fonctionnalité

- Fournit un proxy aléatoire parmi une liste prédéfinie.
- Réponse simple au format JSON, prête à l’emploi.
